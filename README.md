# Rule-based-chatbot
#A rule based chatbot developed using python. It interacts with humans, answer some questions and perform some basic calculations and can also play game with the user.

##Concept used:

.  Python basics

. variables

. conditional statements(if,elif,else)

. string manipulation(.lower)

. while loop

. Membership operator(in)

. Arithmetic operators(+,-,*,/)

. Logical opoerator(or)

. Comparison operator(!=)

. Libraries(random,datetime,zoneinfo)

##Features of chatbot:

 Conversation with the user(Greetings)
 
 Play Game(Number guessing game)
 
 Perform calculation
 
 Study Tips
 
 Time Management
 
 Tell time and date
 
 Exit commands
